
	Напишите в терминале "$npm install" чтобы установить модули.

	Чтобы запустить проект, напишите "npm start" (если установлен angular-cli то можно ng serve). Проект запустится на хосте "http://localhost:4200".

	В данном проекте для позиционирование и сеточной системы используется angular/flex-layout.  

	Когда вы нажимаете на фильм вас переносит на компонент MoviesDetailComponent и в адресной строке передается id каждого фильма (Пример: "http://localhost:4200/movie/1").
	
	В компоненте MoviesList используется библиотека Angular Material Design для таблицы, фильтрации по атрибутам, пагинации. А также если кликнете на название фильма вас также перенесет в MovieDetailComponent.
	
	Данные с файла "mock.data-data.ts" парсились напрямую, можно было в MovieService использовать методы http.get через HttpClientModule, но я использовал на прямую так как быстрее и меньше лишнего кода.
